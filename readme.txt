HTML - Esse código define o tipo de documento, a linguagem, o título, e inclui os arquivos de estilo e de script que serão usados. Note 
que será usado o Bootstrap, um framework que facilita a criação de páginas responsivas e compatíveis com diferentes dispositivos e 
navegadores. Esse código define o cabeçalho, o conteúdo e o rodapé da página. No cabeçalho, temos o logo da página e a busca por 
palavra-chave. No conteúdo, temos as notícias da página, que serão geradas dinamicamente pelo JS. No rodapé, há uma mensagem de 
direitos autorais.

CSS - Esse código define as cores das categorias, o estilo do cabeçalho, do conteúdo e do rodapé da página. Note que está sendo usano 
flexbox, uma técnica que permite alinhar e distribuir os elementos de forma responsiva.

JavaScript - Esse código define as funções para buscar uma notícia por palavra-chave, de exibir as notícias na página ao carregar o 
documento, compartilhar e comentar as notícias, e entre outras pequenas funções.
